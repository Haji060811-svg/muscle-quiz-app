# 筋肉クイズアプリ

解剖学・運動学の「筋肉の起始・停止・作用」を学ぶための4択クイズWebアプリ。

## 現状

- `index.html` 1ファイルで完結(HTML+CSS+JS、外部依存なし)
- 47筋を収録(頸部・肩甲帯・上腕・前腕・体幹・股関節/大腿・下腿)
- 起始・停止をセットで問う4択形式、不正解選択肢は他の筋肉のデータから自動生成
- 成績・間違えた筋肉の記録は `localStorage` にブラウザ内保存(サーバー送信なし)
- 部位別フィルター(すべて/肩/頸部/上腕/前腕/体幹/股関節・大腿/下腿・足)
- 直近10問と同じ筋肉が連続で出ないようローテーション
- PWA対応(`manifest.json` + `sw.js`)。HTTPS/localhostで配信すればホーム画面に追加してアプリのように起動できる(オフラインキャッシュ付き)

データは `index.html` 内の `MUSCLES` 配列(JSオブジェクトの配列)にまとまっている。
各筋肉は `{name, region, origin, insertion, action, hint}` の形式。

## 使い方(現状)

`index.html` をブラウザ(Safari含む)で直接開くだけで動作する。ローカルファイルとして開いても問題なし。
ただしService Workerはセキュアコンテキスト(HTTPSまたはlocalhost)が必要なため、`file://`で直接開いた場合はPWA機能(オフラインキャッシュ・ホーム画面追加)は有効にならない。`npx serve .` などローカルサーバー経由、または後述のホスティング後に有効になる。

## やりたいこと(次のアップデート候補)

- **ホスティング**: GitHub Pages / Vercel / Netlify などに公開し、iPhoneからURLで直接アクセスできるようにする(PWAとして完全に機能させるために必須)
- **プッシュ通知**: PWA + Web Push API(要バックエンド、またはOneSignal等のサービス)で1日3回のリマインダーを送る
- **クロスデバイス同期**: 現状はブラウザごとのlocalStorageのみ。Supabase等の簡易バックエンドでスコアをアカウント単位に同期する
- **問題の追加**: 顔面筋・手内在筋・足内在筋など、より細かい筋肉の追加
- **出題形式の拡張**: 「作用」だけを問う問題、画像(筋肉の図)を使った問題など
