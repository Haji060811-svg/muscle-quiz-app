# 筋肉クイズアプリ

解剖学・運動学の「筋肉の起始・停止・作用」を学ぶための4択クイズWebアプリ。

## 現状

- `index.html` 1ファイルで完結(HTML+CSS+JS、外部依存なし)
- 63筋を収録(頸部・肩甲帯・上腕・前腕・体幹・股関節/大腿・下腿)。筋群(内転筋群・大腿四頭筋・ハムストリングスなど)は個々の筋肉に分割して収録
- 3モード: 「今日の10問」(1日10問の自動出題+進捗表示)/「自由演習」(部位フィルター付き4択)/「暗記カード」(自己採点式)
- 自由演習・今日の10問では出題形式(起始のみ・停止のみ・作用のみ・起始停止セット・名称当て)がランダムに変わり、同じ筋肉でも様々な角度から復習できる
- 簡易間隔反復: 間違えた筋肉・出題回数が少ない筋肉ほど優先的に出題される重み付き抽選
- 部位の簡易シルエット図を表示(起始・停止のピンポイント図ではなく、大まかな部位のハイライト)
- 成績・間違えた記録・習熟度は `localStorage` にブラウザ内保存(サーバー送信なし)
- アプリ内リマインダー(通知許可時、開いたタイミングで「今日の10問」未完了を通知。バックグラウンドでの自動配信ではない)
- PWA対応(`manifest.json` + `sw.js`)。HTTPS/localhostで配信すればホーム画面に追加してアプリのように起動できる(オフラインキャッシュ付き)

データは `index.html` 内の `MUSCLES` 配列(JSオブジェクトの配列)にまとまっている。
各筋肉は `{name, region, origin, insertion, action, hint}` の形式。

## 使い方(現状)

`index.html` をブラウザ(Safari含む)で直接開くだけで動作する。ローカルファイルとして開いても問題なし。
ただしService Workerはセキュアコンテキスト(HTTPSまたはlocalhost)が必要なため、`file://`で直接開いた場合はPWA機能(オフラインキャッシュ・ホーム画面追加)は有効にならない。`npx serve .` などローカルサーバー経由、または後述のホスティング後に有効になる。

現在、GitHub Pages( https://haji060811-svg.github.io/muscle-quiz-app/ )で公開済み。

## やりたいこと(次のアップデート候補)

- **本格的なプッシュ通知**: 現状はアプリを開いた時のみのアプリ内リマインダー。アプリを閉じていても届く通知には、Web Push API用のバックエンド(サーバーで定時配信、またはOneSignal等のサービス)が必要
- **クロスデバイス同期**: 現状はブラウザごとのlocalStorageのみ。Supabase等の簡易バックエンドでスコアをアカウント単位に同期する
- **問題の追加**: 顔面筋・手内在筋・足内在筋など、より細かい筋肉の追加
- **図解の精密化**: 現状は部位を大まかにハイライトする簡易シルエット図。起始・停止のピンポイントを示す本格的な筋肉図にする場合は監修が必要
